关于中国知网爬虫代码：

1.注意: 代码主体来源为https://blog.csdn.net/weixin_46266590/article/details/113076090 我只是做了简单的改动！

2.运行代码前一定要先安装好相关库（代码第1-3行和71-74行），尤其是selenium，安装方法可自行搜索。

3.为了确保代码正常运行，需要提前建立好一个txt文件（存储所有文献的网址）和一个xls文件（存储最终爬取的文献摘要等内容），具体可看第45行和第133行的代码注释。

4.需要修改的参数主要有4个：第23和24行的关键词一和关键词二，第49行的爬取页数（也就是需要爬取的文章个数）和第130行的需要获取摘要等详情内容的文献个数，可根据自己的需求更改。

5.该代码爬取文献标题、作者、摘要、关键词、期刊类型，并归类到Excel文件当中，同时判断文献是否为北大核心或CSSCI。

关于万方数据爬虫代码：

1.代码主体来源为https://blog.csdn.net/m0_43450539/article/details/115999317 已做简单改动。

2.运行代码前一定要先安装好相关库（代码第1-3行和71-74行），尤其是selenium，安装方法可自行搜索。

3.代码运行后需要手动输入一次检索词，需要爬取得文章数量可自行更改。

4.输出结果文件以json格式保存。

5.如果爬取途中意外中断，可能是chrome 崩溃或万方反爬机制的原因，可尝试修改间隔时间（已在代码中注释），并尝试从中断的那一页继续爬取（即修改页码，具体修改哪一行已在代码中注释）
